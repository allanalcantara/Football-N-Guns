-- fb_ukfootie2 by rtxA --

Map created specifically to use with my new mini-mod Football N' Guns. Check it out in https://github.com/rtxa/football-n-guns

# Game mode description:
- To score enter the ball to the enemy's goal.
- There are four classes: Scout, Medic, Soldier and Goalkeeper. Each one has its advantages and disadvantages. 
- Goalies have invulnerability and quad damage but can not leave the goal area.

# Notes:
- Map source has been included so you can create your own maps for this game mode.
